# Описание компонентов сушностей BIM-приложения на Unity

## 1. XML
### 1.1. Словарь XML сущьностей
Document - основной класс, содержащий в себе все сведения о проекте;
Default - перечень значений по умолчанию для свойств элементов разных категорий для всех элементов основы;
Element - базовый класс для всех элементов модели;
Site - площадка, на которой распологаются сооружения;
Building - сооружение или его часть, имеет собственные границы и является основой для пространств;
InsertPoint - точка вставки элемента в локальных координатах основы. Задает начало локальных координат элемента;
Space - пространство, строительный объем, ограниченный границами и разделителями основы;
Wall - стена;
ExtrusionGeometry - геометрия заданная выдавливанием контура;
Loop - замкнутый плоский контур;
CheckPoint - точка задающая начало контура и его свойства;
