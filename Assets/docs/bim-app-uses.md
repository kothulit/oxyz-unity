# Возможности, описание и порядок их применения в BIM-приложении на Unity

## 1. Возможности на этапе MVP
### 1.1. Загрузка и создание проекта
- нажимаю в меню сверху посередине выпадающую вкладку "Файл"
- во вкладке три команды для работы с фалом проекта: "Создать", "Открыть", "Сохранить" (неактивная)
- нажимаю на "Создать", появляется окно проводника (браузера) с выбором папки для проекта и именем проекта (папки проекта)
- после выбора папки и названия нажимаю "Создать" внизу эрана, или нажимаю кнопку "Отмена", которая прервет команду.
